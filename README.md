# Portfolio

A modern and interactive developer portfolio built to showcase my projects, technical skills, development journey, and creative approach toward software engineering.

---

## About This Project

This portfolio was designed and developed to represent my journey as a Computer Science student and full-stack developer. The goal was to create a smooth, modern, and immersive experience that reflects both my technical abilities and creative mindset.

The project focuses on:
- Building scalable and responsive interfaces
- Creating engaging user experiences
- Showcasing real-world projects and skills
- Sharing engineering thoughts and development insights
- Combining clean design with modern frontend development

---

## Features

- Modern responsive design
- Interactive UI animations
- Smooth scrolling experience
- Project showcase section
- Technical skills and tools
- Engineering blogs and insights
- Leadership and extracurricular highlights
- Optimized for different screen sizes

---

## Tech Stack

Built using:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
